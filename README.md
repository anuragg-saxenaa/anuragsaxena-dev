## Getting Started

To run this project locally:

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```
2. Navigate to the project directory:
   ```sh
   cd saxena-dev-profile-page
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
